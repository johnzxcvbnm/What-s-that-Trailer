About

What's That Trailer (WTT) is a passion project created by John Kusching, Kate Weingart and Steve Nepote.

You can view the project README here

Movies are announced long before they are released with exciting trailers coming out as far out as a year before release date. With all the bustle of everyday life, often times we forget about these exciting movies. WTT is a single-page application that helps solve the problem of forgetfulness by letting registered users add movies directly onto their user page.

Users can:

  • Add movies to the WTT database
  • Edit and remove movies on the WTT database
  • Add movies to their user page from the WTT database
  • Add movies to their user page from OMDB (a seperate online database)
  • Remove movies from their user page
